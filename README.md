# Crypto Dashboard

## Project Overview
Crypto Dashboard is a **modern web application** that allows users to **track cryptocurrencies, view their portfolio, analyze coin trends, and manage alerts**. It provides **real-time data**, **interactive charts**, and a **user-friendly interface**. Users can **search, sort, and filter coins**, view their **portfolio performance**, and stay updated with **price alerts**.  

The project focuses on building a **responsive, mobile-friendly crypto tracking platform** with **dark mode support** and **high performance**.  

---

## Features

- **Live Cryptocurrency Tracking** – View real-time prices, market cap, and volume.  
- **Portfolio Management** – Add coins to your portfolio and track profits/losses.  
- **Search & Sort** – Quickly find coins using search and sorting tools.  
- **Interactive Charts** – Visualize price trends over time.  
- **Price Alerts** – Set custom alerts for coins to stay updated.  
- **Responsive Design** – Works on desktop, tablet, and mobile devices.  
- **User-Friendly Interface** – Easy navigation with tabs, tables, and cards.  

---

## Tools & Technologies Used

- **Frontend**:  
  - **React.js** – For building interactive UI components.  
  - **HTML5 & CSS3** – Styling and structure.  
  - **JavaScript (ES6+)** – For logic and interactivity.  
- **Charts**:  
  - **Chart.js or Recharts** – For interactive crypto price charts.  
- **Data**:  
  - **CoinGecko API** – Real-time cryptocurrency data.  
- **State Management**:  
  - **React useState & useEffect hooks** – Local state handling.
